# SOS Point Ordinateur Site Web

## Présentation

Ce projet est un site web réalisé dans le cadre du BTS SIO option SLAM.
Il s’agit d’un site pour une entreprise d'informatique

Le site permet de présenter l’activité de l’entreprise, ses services, ainsi que de proposer un formulaire de contact permettant aux clients d’envoyer un message.

---

## Objectifs

* Crée un site web
* Mettre en pratique les bases du développement web (HTML, CSS, PHP)
* Utiliser une base de données pour enregistrer des informations
* Mettre en place un  admin

---

## Compétences utilisées

* HTML
* CSS
* PHP (procédural)
* MySQL (PDO)

---

## Fonctionnalités

### Site public

* Page d’accueil avec présentation de l’entreprise
* Page des services proposés
* Page de contact avec formulaire
* Page à propos
* Intégration d’une carte Google Maps

### Formulaire de contact

* Enregistrement des messages en base de données
* Affichage d’un message de confirmation après envoi

### Espace administrateur

* Connexion avec identifiant et mot de passe
* Consultation des messages reçus
* Visualisation du détail d’un message
* Suppression des messages
* Gestion du statut (lu / non lu)

## Base de données

'sos_point_ordinateur_site'

* Le fichier SQL est disponible dans le dossier : database/sos_point_ordinateur.sql
